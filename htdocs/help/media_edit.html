Edit Media

If you need to make any changes to your media, whether to the actual file or the default caption, you will make these modifications on the Edit Media page.

There are three parts of the Edit Media page:

  1. Properties
  2. Content
  3. Actions

Properties

The top area on the Edit Media page has the media's properties. This information is not editable.

  • Media Id: Set by the CMS on creation of new media.
  • Current Version: Field set by the CMS. This reflects the version number of the media.
  • Published Version: Version of the media that was last published. This may be different from the current version if changes were made to the media after the last publish. This field is set automatically by the CMS.
  • URL: The URL field is also controlled by the CMS. Although it appears as a non-editable field, it can be changed by modifying the fields that make up the URL. These fields are the site and category.
  • Creation Date: The date your media was created.

Content

Below the Properties is the Content. This is where you specify information about your media, such as its Site/Category and default caption.

  • Type: Each media file has a Type. This is set by the CMS on creation of new media and is not an editable field. If you selected the incorrect Media Type, you will need to return to the New Media page. Media Type selection determines the attributes available to the media.
  • Title: Required field used to give your media file a unique name.
  • Site/Category: Required field used to designate the URL of the site or category where you want to upload your media. Clicking the Choose button will launch a menu tree where you can find the correct category. From the menu tree click OK to accept your choice or Cancel to return to the New Media page.
  • Upload File: If you would like to replace the current media content with a new file, click on Browse and find an item to upload.
  • Default Caption: Text field where you can enter a default caption to associate with your media. This is an optional field and may not appear within the output depending on the template configuration.
  • Default Copyright: Text field where you can enter a default copyright to associate with your media. This is an optional field and may not appear within the output depending on the template configuration.
  • Alt Tag: Text field where you can enter an alt tag to associate with your media. Alt tags are displayed on a web page when the image is not available, or spoken by a screen reader.
  • Notes: Optional field where you can enter information about the media that will not be displayed in the story.
  • Version: You can revert to a previously saved version. Select the version number from the drop-down and click on Revert. If you prefer, you can also View a previously saved version.
  • Contributors: A contributor can be any person, for example a writer or photographer who contributed to the media. Click on Edit Contributors to associate, or un-associate your contributors. See Associate Contributors for additional help.
  • Content: Starting with Krang v3.01, media files consisting of text content can be edited online rather than uploaded. (Images and other types of binary media will not show this field.)

Actions

At the bottom of the Edit Media page, there are buttons and different actions to choose from.

  • Cancel: Click on Cancel to discard any unsaved changes.
  • Save & Stay: Click on Save & Stay to save your media, publish it to preview, and remain on the Edit Media screen.
  • Save: Click on Save to save your media and return to your Workspace.
  • Publish to Preview: Click on Publish to Preview to publish your media immediately to the preview site. (Starting with Krang v3.01, clicking Save & Stay also triggers an automatic Publish-to-Preview.)
  • Publish: Click on Publish to publish your media immediately to the live site. If you would like to schedule your media to publish at a later date, click on Schedule at the top of the page. See Schedule Editor for more information.
  • Delete: Click on Delete to delete your media from the CMS.
    In addition to the action buttons featured at the bottom of the page, the following two actions are available at the top:
  • Log: Please see Log for more information.
  • Schedule: Please see Schedule for more information.
