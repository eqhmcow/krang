Krang VS Bricolage Comparison

Krang was designed as a replacement for our usage of Bricolage. During the design of Krang features not found in Bricolage were added and some Bricolage features were removed. This page provides a summary of the most important of these changes, as well as some examples of things that stayed the same in Krang.
