User Management

This is where you create user accounts and establish permission levels. A user's permission set is determined by his or her association to a specific permission Group. For example, if you want user Sally Jones to be allowed to edit media, you'll go to her User Profile page and enter her into a Group that has Media editing permissions.

Within the User Management screen, you can also:

Search:Searches users by last name or first name. Do a blank search to list all users.
Add: Click Add a User to create a new user account. It brings you to the User Profile page. Enter the requested information and then click Save at the bottom of the page.
Edit:Click Edit to modify a person's User Profile.
Delete:To delete a user account, check the box, scroll to the bottom of the page and click the Delete Checked button.
