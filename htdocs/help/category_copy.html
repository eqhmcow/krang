Copy Category

Here you may copy the content of one category and its subcategories to another category and determine what exactly you want to copy. The category you copy from has been selected when you clicked on the Copy button on the previous screen. Next you have to determine where you want to copy to and what you want to copy along with the subcategories.

  • Category Copying To: Using the usual category chooser, select the category you want to copy to.
  • Copy Stories? Check this option if the copy should include stories living below Category Copying From.
  • Copy Media? Check this option if the copy should include media living below Category Copying From.
  • Copy Templates? Check this option if the copy should include templates living below Category Copying From.

The Overwrite Existing Files checkbox modifies the copy behavior:

  • Without Overwrite: If the URL of a would-be-copied Story, Media or Template is already occupied by some other Story, Media or Template, Krang will ask you whether you want to cancel the whole copy operation or copy at least the non-conflicting Stories, Media and Templates.
  • With Overwrite: In this case, Krang will move conflicting Stories, Media and Templates to the Trash, in order to make place for the copy.
