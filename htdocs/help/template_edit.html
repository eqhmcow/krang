Edit Template

If you need to make any changes to your template, you will make these modifications on the Edit Template screen.

The Edit Template Screen has the following editable fields:

  • Upload File: Click on Browse to find a file on your computer. This is not required, as the emplate can be edited directly in the CMS (see Content).
  • All Versions: You can revert to a previously saved version. Select the version number from the drop-down and click on Revert. If you prefer, you can also View a previously saved version.
  • Content: You can edit your template directly in the CMS. Use this text box for template code.
  • Test: Check this box and you can test the template without deploying it and affecting stories being published.

Actions

At the bottom of the Edit Template page, there are buttons and different actions to choose from.

  • Cancel: Click on Cancel to discard any unsaved changes.
  • Save & Stay: Click on Save & Stay to save your template and remain on the Edit Template screen.
  • Save: Click on Save to save your template and return to your Workspace.
  • Deploy: Click on Deploy to immediately deploy your template. Templates that are deployed will be used to format Stories that are published.
  • Delete: Click on Delete to delete your template from the CMS.
