Associate Contributor

Each Story/Media can be associated with a contributor(s). These contributors can be writers, photographers, etc. Entering Contributors into the system is a separate process. Please refer to Contributors for more information. After clicking Edit Contributors on the Edit Story/Media screen, to associate a contributor, check the box next to the Contributor you want to associate, then click the Associate button. Click Return to return to the Edit Story/Media page.

In addition to associating a contributor, you can also Search for a contributor, Add a contributor, Reorder or Un-associate the contributor.

Search: Search by First Name or Last Name. Click Search for results.
Add a Contributor: If the Contributor you are looking for is not in the CMS yet, you can add it here. Click on Add a Contributor. Fill in the text fields and click Save. For more information, please see Add Contributor.
Reorder: Contributors can be re-ordered to appear in a specific order. Use the pulldowns to adjust the Position numbers, then click the Reorder button.
Un-Associate: If the incorrect contributor was associated with a Story, you can un-associate. Check the box next to the contributor and click Unassociate.
