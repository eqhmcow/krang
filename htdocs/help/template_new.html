New Template

The New Template page is used to create new templates. Templates format the content in your CMS with HTML and other formatting languages.

To create a new template, enter the following fields:

Element: Select an element from the drop-down. The element should be related to the template. This is an optional field: you can create a template without selecting an element.
Filename: Name your template file. This field is not required if you upload a template file.
Site/Category: This is a required field used to designate the URL of the site or category where you want to upload/create your template. Clicking the Choose button will launch a menu tree where you can find the correct category. From the menu tree click OK to accept your choice or Cancel to return to the New Template page.
Upload File: Click on Browse to find a file on your computer. This is not required, as the template can be created directly in the CMS (see Content).
Content: You can create your template directly in the CMS. Use this text box for template code.
Test: Check this box and you can test the template without deploying it and affecting stories being published.

After entering in the above information click on one of the Buttons at the bottom of the page. Cancel will discard your new template. Save & Stay will save your template and refresh the current page. Save will save your template and send you to your Workspace.

Deploy will deploy the template to production. It is suggested that you test with your template before deploying.
