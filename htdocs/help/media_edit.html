Edit Media

If you need to make any changes to your Media, whether it is the actual file or changing the default caption you will make these modifications on the Edit Media screen.

On the edit Media page, there are three parts:

  1. Properties
  2. Content
  3. Actions

Properties

The top area on the Edit Media page has the Media's properties. This information is not editable.

  • Id: Media ID. Set by the CMS on creation of a New Media. This field cannot be changed.
  • Current Version: Field set by the CMS. This is a non-editable field, reflects the version number of the Media.
  • Published Version: Version of the Media that was last published. This may be different from the current version if changes were made to the Media after the last publish. This field is controlled automatically by the CMS.
  • URL: The URL field is controlled by the CMS. Although it appears as a non-editable field, it can be changed by modifying the fields that make up the URL. These fields are the site and category.
  • Creation Date: The date your Media was created. This field is set by the CMS and is not editable.

Content

Below the properties, is the content. This is where you specify information about your Media, such as its Site/Category and default caption.

  • Type: Each Media has a Type. This is set by the CMS on creation of a New Media and is not an editable field. If you selected the incorrect Media Type, you will need to return to the New Media page. Media Type selection determines the attributes available to the Media.
  • Title: Required field used to give your image a unique name.
  • Site/Categories: Required field used to designate the URL of the site or category where you want to upload your media. Clicking the Choose button will launch a menu tree where you can find the correct category. From the menu tree click OK to accept your choice and close the function or Cancel to return to the New Media page.
  • Upload File: Click on Browse to find an image or other media you would like to upload. This is a required field.
  • Default Caption: Text field where you can enter a default caption to associate with your media. This is an optional field and may not appear within the output depending on the template configuration.
  • Default Copyright: Text field where you can enter a default copyright to associate with your media. This is an optional field and may not appear within the output depending on the template configuration.
  • Alt Tag: Text field where you can enter an alt tag to associate with your media. Alt tags are displayed on a web page when the image is not available or spoken by a screen reader.
  • Notes: Optional field where you can enter information about the image that will not be displayed in the story.
  • Contributors: A contributor can be any person, ex. Writer or photographer who contributed to the Media. Click on Edit Contributors to associate, or un-associate your contributors. See Associate Contributors for additional help.
  • Content: Starting with Krang v3.01, media files consisting of text content can be edited online rather than uploaded. (Images and other types of media will not show this field.)

Actions

At the bottom of the Edit Media page, there are buttons and different actions to choose from.

  • Revert: You can revert to a previously saved version. Select the version number from the drop-down and click on Revert. If you prefer, you can also View a previously saved version.
  • Save: Click on Save to save your Media. Clicking on this button will take you up a level to 'My Workspace'.
  • Save & Stay: Click on Save & Stay to save your Media and remain on the Edit Media screen.
  • Publish: Click on Publish and your Media is published immediately. If you would like to schedule your Media to publish at a later date. Click on Schedule at the top of the page. See Schedule Editor for more information.
  • Reset: Clicking on Reset will reset all text fields to last saved version.
  • Delete: Click on Delete to delete your Media from the CMS.
    In addition to the action buttons featured at the bottom of the page, the following two actions are available at the top:
  • Log: Please see Log for more information.
  • Schedule: Please see Schedule for more information.
