My Alerts

You can alert fellow users of your CMS to certain actions by setting up an Alert. Alerts will send an email to a user if the action specified occurs. The different actions that alerts can be set for are:

    Check In
  • Check Out
  • Move To
  • New
  • Publish
  • Save

An alert for one of these actions can be set for specific Story or Media objects, for all Story or Media objects within a certain category, or for every object in the system (including templates). To set up a new Alert, follow these steps:

  1. Select an Action from the dropdown list.
  2. If you wish to limit the alert to a specific Story/Media object, select the Object Type & Object ID.
  3. If you wish to limit the alert to Story/Media objects beneath a certain category, choose a Site/Category.
  4. If you wish to limit the alert to Stories that are moved to a given desk, select a Desk.
  5. Click on Add Alert.

To delete, edit, or create new alerts at any time, click on My Alerts in the left navigation.
