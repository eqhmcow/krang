Group Management

Groups are sets of permissions. When a user is assigned to a certain Group, the user inherits that Group's permissions. You can also create new Groups and edit the permissions of existing ones.

  • Search: Searches Groups by name. Do a blank search to list all Groups.
  • Add: Click on Add a Group to create a new Group. It brings you to the Group Add page, where you will enter the Group's name and the permissions within that group.
  • Edit:Click Edit to modify a Group's permissions.
  • Delete:To delete a Group, check the check box, scroll to the bottom of the page and click the Delete Checked button.
