=head1 Krang Requirements

The following packages and modules must be installed prior to
installing Krang.

=head2 Software Packages

=over 4

=item Perl 5.6.1

=item Apache 1.3.27

=item mod_perl 1.27

=item MySQL 3.23.53

=item ImageMagick-5.5.3

1) Download the source: 

  wget ftp://ftp.rge.com/pub/multimedia/imagemagick/ImageMagick-5.5.3-2.tar.gz

2) Decompress:

  tar zxvf ImageMagick-5.5.3-2.tar.gz
  cd ImageMagick-5.5.3

3) Build and install:

  ./configure --without-perl
  make

4) Build and install the Perl module (must be done as a separate step)

  cd PerlMagick
  perl Makefile.PL
  make
  make install

=back

=head2 Perl Modules

All Perl modules used by Krang and contained in ext-src/ and are
installed by running C<make>.
