Introduction

Krang is a web-accessible application that will provide your staff with the ability to easily create, manage and publish website and email content. Krang creates a website that can be tailored to your specific needs, enabling you to provide the right content to your target audience.

To use Krang, you don't need to be an HTML producer or programmer - you need only a basic understanding of Internet usage. It is recommended that new users contact the site administrator for Krang training first, and then refer to the manual as questions arise.

System Requirements

Web browsers vary in their support for current recommendations and standards relating to the presentation and utility of Web content. Application of these standards within Krang excludes less sophisticated browsers from access to its full functionality. For best results, the following browsers are recommended:

  • Microsoft Internet Explorer, version 6.0+ (Windows)
  • Firefox, version 1.5+ (Windows / Linux / Mac OS X)
  • Safari, version 1.2+ (Mac OS X)
