Site Management

A Site is your web site where content items are published. In Krang, you must first create a Site, then create Categories within a Site. Sites are usually created by a CMS administrator.

The site admin defaults to the Site List page. This page lists all the Categories in the system. Additionally, you can:

    Search:Search for a Site by site name. Add:Click on Add A Site, enter in the requested information and click Save to save your Site. Edit: Click on Edit to make changes to your Site. Delete: To delete a Site, check the check box and click on Delete Checked.
