My Alerts

You can alert fellow users of your CMS to certain actions by setting up an Alert. Alerts will send an email to a user if the action specified occurs. The different Actions that alerts can be set for are:

  • Check In
  • Check Out
  • Move To
  • New
  • Publish
  • Save
An alert for one of these actions can be set for Stories, Media, and Templates within a category that you choose during your set up of the Alert. To set up a new Alert, follow these steps:

1) Select an Action
a. From the drop-down menu, select an action from the list. Actions include: Check In, Check Out, Move To, New, Publish, Save
2) Choose a Site/Category
3) Select a Desk (this is an optional field. Some actions do not require a desk).
4) Click on Add Alert

To delete, edit, or create new alerts at any time, click on My Alerts from the Left Navigation.
