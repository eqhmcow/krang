Category Management

A Category is the place on your site where content items are published. In Krang, you must first create a Site, then you can create Categories within a Site. You can create and edit new Categories in this area. Publishing content items in multiple Categories can help to organize your site's components.

The category admin defaults to the Category List page. This page lists all the Categories in the system. Additionally, you can:

  • Search: Search for a Category by category name.
  • Add: Click on Add A Category, enter in the requested information and click Save to save your Category.
  • Edit:Click on Edit to make changes to your Category.
  • Copy:Click on Copy to duplicate some or all of an existing Category.
  • Delete:To delete a Category, check the check box and click on Delete Checked.
